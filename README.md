## 🎨 Customization

### Colors
Modify the CSS variables in `styles.css`:
```css
:root {
    --primary: #6F3AFF;
    --secondary: #2A1454;
    --accent: #FF6B6B;
    /* ... more variables */
}
```

### Content
Update the HTML content in `index.html` to match your information.

### Styling
Modify `styles.css` to change layouts, animations, and visual effects.
